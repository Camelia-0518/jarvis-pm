## 🎯 核心功能展示

### 1. PRD质量评分系统
```python
POST /api/v1/evaluation/score
Response: {
  "total": 82,           # 总分
  "grade": "B+",         # 等级
  "breakdown": {
    "completeness": 85,  # 完整性
    "accuracy": 80,      # 准确性
    "usability": 85,     # 可用性
    "compliance": 90,    # 合规性
    "innovation": 70     # 创新性
  }
}
```

### 2. RAG增强生成
```python
POST /api/v1/rag/generate
# 多路召回 → 重排序 → 上下文组装 → LLM生成
```

### 3. Agent市场
```python
GET /api/v1/marketplace/agents      # 浏览Agent
POST /api/v1/marketplace/agents     # 创建Agent
POST /api/v1/marketplace/agents/{id}/review  # Agent协作Review
```

### 4. 代码生成
```python
POST /api/v1/code/prototype         # PRD → 原型
POST /api/v1/code/api-spec          # PRD → API
POST /api/v1/code/components        # PRD → 组件
```
