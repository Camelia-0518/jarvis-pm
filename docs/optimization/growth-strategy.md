## 3. 病毒传播机制

### 3.1 病毒系数（K-factor）设计

**目标**：K-factor > 0.5

**计算公式**：
```
K = 邀请率 × 接受率 × 每个邀请带来用户数

示例：
- 邀请率：20%（每5个用户有1个发起邀请）
- 接受率：30%（被邀请人30%注册）
- 每个邀请带来用户数：1.5（平均每人邀请1.5人）

K = 0.2 × 0.3 × 1.5 = 0.09（当前）

目标优化：
- 邀请率提升至40%
- 接受率提升至50%
- 每个邀请带来用户数提升至2

K = 0.4 × 0.5 × 2 = 0.4（目标）
```

### 3.2 邀请奖励机制

**双边奖励**：

```
邀请人奖励：
├── 每邀请1人：¥20积分
├── 邀请3人：额外1个月专业版
├── 邀请5人：额外3个月专业版
└── 邀请10人：终身专业版

被邀请人奖励：
├── 注册奖励：¥10积分
├── 完成首个PRD：额外¥10积分
└── 付费升级：邀请人获得¥50积分
```

**阶梯奖励可视化**：

```
邀请进度：
[████░░░░░░] 2/10

奖励解锁：
✅ 邀请1人：¥20积分
✅ 邀请2人：¥20积分
🎁 邀请3人：+1个月专业版
⬜ 邀请5人：+3个月专业版
⬜ 邀请10人：终身专业版
```

### 3.3 病毒产品机制

**协作型病毒**：

```
场景：用户A创建PRD，需要用户B评审

流程：
1. 用户A分享PRD链接给用户B
2. 用户B打开链接，看到PRD内容
3. 提示"注册后可添加评论"
4. 用户B注册，成为新用户
5. 用户B的PRD也会分享给其他人
```

**内容型病毒**：

```
场景：用户导出PRD分享给团队

流程：
1. 用户导出PRD（Markdown/PDF）
2. 文档底部带品牌信息
3. 团队成员看到品牌信息
4. 感兴趣的人访问官网
5. 转化为新用户
```

**模板型病毒**：

```
场景：用户在模板市场发现好模板

流程：
1. 用户浏览模板市场
2. 发现优质模板
3. 分享给同事/朋友
4. 被分享人注册使用
5. 优质模板作者获得奖励
```

## 4. 用户激活策略

### 4.1 激活定义

**激活行为**：完成首个PRD并导出

**激活指标**：激活率 > 60%

### 4.2 新手引导流程

** onboarding 流程设计**：

```
Step 1: 欢迎页（30秒）
├── 欢迎语："欢迎来到Jarvis PM"
├── 价值展示："10分钟完成专业PRD"
└── CTA："创建你的第一个PRD"

Step 2: 选择模板（1分钟）
├── 推荐模板（基于行业）
├── 热门模板
└── 空白创建

Step 3: AI对话引导（5分钟）
├── AI问候："我是你的AI产品助手"
├── 提问："请简单描述你的需求"
├── 多轮对话收集信息
└── 生成PRD大纲

Step 4: 编辑完善（3分钟）
├── 展示生成的PRD
├── 提示可编辑
└── 提供优化建议

Step 5: 导出分享（1分钟）
├── 导出成功庆祝
├── 展示节省的时间
└── 提示分享给团队
```

### 4.3 激活优化策略

| 策略 | 具体行动 | 预期效果 |
|:-----|:---------|:---------|
| **降低门槛** | 提供示例PRD一键复制 | 提升30%启动率 |
| **即时价值** | 首屏展示"为你节省X小时" | 提升价值感知 |
| **进度展示** | 显示PRD完成进度 | 提升完成率 |
| **成功庆祝** | 完成首个PRD后庆祝动画 | 提升成就感 |
| **社交证明** | 显示"X位前端工程师正在使用" | 提升信任 |
