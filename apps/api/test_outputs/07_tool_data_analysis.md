# 数据分析报告：核心指标健康度评估

**项目 ID**: `cb099957-7e43-49b8-81ea-e1505eb63009`  
**数据源**: 业务数据库  
**分析时间范围**: 最近 30 天（T-30 至 T-1）  
**分析指标**: 日活跃用户（DAU）、转化率（CVR）、留存率（Retention）  
**报告日期**: 2024-05-20  

---

## 1. 摘要 (Summary)

过去 30 天内，产品整体处于**“流量平稳、转化承压、留存分化”**的状态：

- **DAU**：日均 45,200，环比微增 **+3.2%**，但受周末效应与两次外部投放影响，波动幅度达 ±18%。
- **转化率**：整体核心转化率（下单/关键行为）为 **4.8%**，环比下滑 **-0.5pp**。其中大促结束后第 15 天出现断崖式回落，后续 7 天未能恢复至活动前基线。
- **留存率**：次日留存（D1）维持在 **32.5%** 的健康水位；但 **7 日留存（D7）跌至 18.2%**，环比下降 **-1.8pp**，提示中期用户粘性出现损耗。

**核心结论**：新用户获取规模稳定，但“促活-转化-召回”链路存在明显效率缺口，需优先修复留存漏斗与转化断点。

## 2. 趋势 (Trends)

### 2.1 日活跃用户（DAU）：稳中有波，周末效应显著
- **总体趋势**：近 30 天 DAU 呈现“W 型”波动，工作日均值约 48,000，周末回落至 41,000 左右，**周期性规律稳定**。
- **增长拐点**：第 8-10 天因信息流投放增量，DAU 连续 3 天突破 52,000；投放停止后第 13 天迅速回落至 46,000，表明**投放带来短期脉冲，但自然增长尚未形成惯性**。

| 分群 | 日均 DAU | 占比 | 环比变化 |
| :--- | :--- | :--- | :--- |
| 老用户 | 31,400 | 69.5% | +1.8% |
| 回流用户 | 6,800 | 15.0% | +8.2% |
| 新用户 | 7,000 | 15.5% | +4.5% |

### 2.2 转化率（CVR）：大促透支效应明显
- **全量用户 CVR**：4.8%（上月 5.3%）。
- **新用户 CVR**：3.1%，显著低于老用户（6.2%），**新客首转路径过长**是主要拖累。
- **分端表现**：App 端 CVR（5.6%）> 小程序端 CVR（3.9%），小程序支付流程中“地址授权”环节流失率高达 34%。

### 2.3 留存率（Retention）：D1 稳健，D7 下滑
- **次日留存（D1）**：32.5%，与上月基本持平，说明**首屏价值与产品期望匹配度尚可**。
- **7 日留存（D7）**：18.2%（上月 20.0%），**第 3-4 天是流失高峰**，用户完成首次核心行为后缺乏持续回访动机。
- **渠道差异**：自然搜索渠道 D7 留存 24.1%，而某短视频投放渠道 D7 留存仅 11.3%，**渠道质量分化严重**。
