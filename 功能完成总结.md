## 🚀 启动方式

### 方式1: 双击启动（推荐）

```bash
双击 "启动JarvisPM.bat"
```

### 方式2: 手动启动

```bash
# 终端1: 后端
cd apps/api
python main.py

# 终端2: 前端
cd apps/web
npm run dev
```
